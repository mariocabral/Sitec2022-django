# Sitec2022-django


## Guía de uso

```bash
git clone git@github.com:mariocabral/Sitec2022-django.git
cd Sitec2022-django
docker compose up
```

Una vez iniciado acceder a [http://localhost:8000/](http://localhost:8000/)
